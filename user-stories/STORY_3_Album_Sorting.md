# Story 3: Album Sorting

**As an** end user,
**I want** to sort the album catalog by different fields,
**so that** I can quickly find albums by title, artist, year, or genre.

## Acceptance Criteria

### Frontend UI

1. **Sort by title** — Click the "title" link in the "sort by" controls. Albums reorder alphabetically by title in ascending order.
2. **Sort by artist** — Click the "artist" link. Albums reorder alphabetically by artist name.
3. **Sort by year** — Click the "year" link. Albums reorder by release year.
4. **Sort by genre** — Click the "genre" link. Albums reorder alphabetically by genre.
5. **Toggle sort direction** — Click the chevron icon in the sort controls. The icon changes from chevron-up (ascending) to chevron-down (descending) and the sort order reverses. Clicking again toggles back.
6. **Sorting works in both views** — Apply a sort (e.g., by artist descending), then switch from grid to list view. The sort order is preserved in the new view.
