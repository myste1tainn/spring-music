Architecture decision

We chose Strangler Fig + API Façade. The new Go service surfaces the same URL contract — the React UI doesn't know the difference.

Handler → Service → Repository interface. Concrete implementations injected at startup. No SQL in handlers, no HTTP in repos.

The anti-corruption layer lives at the Repository interface boundary — old schema never leaks into new domain types.

The new stack. Both sides still work.

  • ⚛️

  • 🔄
